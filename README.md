<b>Ctrl + Space</b> で デバッグモードの切り替え

### デバッグモード中の操作  
* 0キー：パズルモード切り替え  
* 1キー：当たり判定の 表示/非表示  

### 通常時操作方法
* 決定 / 会話を進める：スペースキー
* キャラクター移動：左右キー
* ジャンプ：上キー
* ポーズ：Pキー
